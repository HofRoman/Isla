# Isla – Islamische App

Eine vollständige islamische Flutter-App mit eleganter schwarz-weißer Design-Sprache und schönen Animationen.

## Features

### 🧠 Quiz – Islamisches Wissen
- 5 Kategorien: Grundlagen, Koran, Propheten, Gebet & Ibadah, Geschichte
- Animierte Fragen und Antworten mit Erklärungen
- Punkte- und Highscore-System mit Konfetti bei Perfektion

### 📖 Koran-Leser
- Suren in Arabisch (originale Schrift)
- **Bubenheim Übersetzung** (Deutsch)
- Englische Übersetzung
- Tabs: Nur Arabisch / Nur Übersetzung / Alle zusammen

### 🕌 Gebet
- Alle 5 Pflichtgebete mit Zeiten und Rakaat-Übersicht
- Wudu-Anleitung Schritt für Schritt
- Dhikr-Duas mit arabischem Text und Bedeutung

### 📿 Dhikr-Zähler (Tasbih)
- 7 Dhikr auswählbar, kreisförmige Fortschrittsanzeige
- Haptisches Feedback, Gesamtzähler gespeichert

### ✨ 99 Namen Allahs (Asma ul-Husna)
- Alle 99 Namen Arabisch / Deutsch / Englisch mit Bedeutung
- Suchfunktion und Detailansicht

## Design
- Weißer Hintergrund, schwarze Elemente, goldene Akzente
- Smooth Animationen (flutter_animate)
- Arabische Schrift, Playfair Display

## Tech Stack
Flutter 3.x · Provider · SharedPreferences · flutter_animate · google_fonts · confetti · percent_indicator

## Installation
```bash
flutter pub get
flutter run
```
